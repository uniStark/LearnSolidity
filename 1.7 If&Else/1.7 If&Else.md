# 1.7 If&Else

在 Solidity 中，`if`、`else if` 和 `else` 用来做条件判断。也就是说，程序会根据不同条件执行不同分支。

这类语句非常常见，比如：

- 判断输入值是否大于某个数字
- 判断一个数字是否为偶数
- 判断某个条件是否满足

## 基本写法

- `if`：当条件成立时执行
- `else if`：前一个条件不成立时，继续判断新条件
- `else`：前面的条件都不成立时执行

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.7;

contract IfElseExample {
    function checkValue(uint256 value) public pure returns (string memory) {
        if (value > 100) {
            return "Value is greater than 100";
        } else if (value == 100) {
            return "Value is exactly 100";
        } else {
            return "Value is less than 100";
        }
    }

    function isEven(uint256 value) public pure returns (bool) {
        return value % 2 == 0;
    }
}
```

## 代码说明

`checkValue(uint256 value)` 会根据输入值返回不同字符串：

- 大于 `100`，返回 `"Value is greater than 100"`
- 等于 `100`，返回 `"Value is exactly 100"`
- 小于 `100`，返回 `"Value is less than 100"`

`isEven(uint256 value)` 则更简单，通过 `%` 取余判断是否为偶数。

如果 `value % 2 == 0`，说明它可以被 2 整除，所以结果是 `true`。

## 小结

条件语句是 Solidity 的基础语法之一。写合约时，只要遇到“如果满足某个条件就执行不同逻辑”的场景，就会用到 `if`、`else if` 和 `else`。
