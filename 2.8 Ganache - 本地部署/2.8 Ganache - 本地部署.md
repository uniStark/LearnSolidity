# 2.8 Ganache - 本地部署

`Ganache` 是一个本地区块链开发工具。它会在你的电脑上启动一条临时的以太坊测试链，方便你本地部署、调试和测试智能合约，而不用真的连接到公网链。

## 安装与启动

```bash
npm install -g ganache
ganache --wallet.totalAccounts 10 --chain.chainId 1337
```

这条命令会启动一条本地链，并预生成 10 个测试账户。

## 默认 RPC 地址

Ganache 启动后，通常会监听：

```text
http://127.0.0.1:8545
```

这个地址就是本地 JSON-RPC 入口。后面用 Remix、Hardhat、ethers.js 或其他工具连接本地链时，通常都要填这个 RPC URL。

## 使用时要注意

- 这是本地开发链，不是真实以太坊网络
- 账户和私钥只是测试用途，不能用于生产环境
- 如果你重启了一条临时本地链，账户、私钥、合约地址和整条链的状态都可能被重置

## 小结

可以把 Ganache 理解成“本地沙盒区块链”：

- 部署快
- 出块快
- 适合学习和调试
