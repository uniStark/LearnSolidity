# 2.4 Struct - 结构体

在 Solidity 中，`struct`（结构体）是一种自定义类型，可以把多个字段组合到一起。它适合表示“一个完整对象”，例如一个人、一件商品或一条订单记录。

## 1. 基本概念

下面定义了一个 `Person` 结构体：

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.7;

contract StructExample {
    struct Person {
        string name;
        uint age;
        address wallet;
    }

    Person public person;

    function setPerson(string memory name, uint age, address wallet) public {
        person = Person(name, age, wallet);
    }
}
```

`Person` 里有三个字段：

- `name`：姓名
- `age`：年龄
- `wallet`：钱包地址

这样我们就能把“一个人的信息”作为一个整体来处理，而不是分成多个互不相关的变量。

## 2. 如何使用 struct

上面例子中的 `setPerson()` 做了两件事：

- 接收姓名、年龄和钱包地址
- 用 `Person(name, age, wallet)` 创建一个结构体并保存

对于初学者来说，可以先把它理解成“按顺序把多个值打包成一个对象”。

## 3. struct 数组

如果要保存很多个人，就可以把结构体放进数组中。

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.7;

contract StructArrayExample {
    struct Person {
        string name;
        uint age;
        address wallet;
    }

    Person[] public people;

    function addPerson(string memory name, uint age, address wallet) public {
        people.push(Person(name, age, wallet));
    }
}
```

这里的重点是：

- `Person[] public people;` 表示“由多个 `Person` 组成的数组”
- `people.push(Person(name, age, wallet));` 表示新增一个人

这种写法非常常见，因为很多业务数据都不是单条，而是一组记录。

## 4. 小结

- `struct` 用来把多个字段组合成一个自定义类型。
- 一个 `struct` 变量可以保存一条完整记录。
- `struct` 也可以放进数组中，表示多条记录。

学会 `struct` 以后，再结合数组和 `mapping`，就能写出更接近真实业务场景的数据结构。
