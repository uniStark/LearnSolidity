# 1.8 Loops

在 Solidity 中，循环可以让一段代码重复执行。最常见的是 `for` 和 `while`。

另外，循环里还有两个常用关键字：

- `break`：立刻结束整个循环
- `continue`：跳过当前这一次循环，直接进入下一次

## 示例合约

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.7;

contract Loops {
    uint[] public list = [1, 2, 3, 4, 5, 6, 7];

    function listWithBreak() public view returns (uint[] memory) {
        uint[] memory numbers = new uint[](list.length);
        for (uint i = 0; i < list.length; i++) {
            if (list[i] == 5) {
                break;
            }
            numbers[i] = list[i];
        }
        return numbers;
    }

    function listWithContinue() public view returns (uint[] memory) {
        uint[] memory numbers = new uint[](list.length);
        for (uint i = 0; i < list.length; i++) {
            if (list[i] == 5) {
                continue;
            }
            numbers[i] = list[i];
        }
        return numbers;
    }

    function listWithWhile() public view returns (uint[] memory) {
        uint[] memory numbers = new uint[](list.length);
        uint i = 0;
        while (i < list.length) {
            numbers[i] = list[i];
            i++;
        }
        return numbers;
    }
}
```

## 代码说明

`listWithBreak()` 使用 `for` 循环遍历数组。当遇到数字 `5` 时，执行 `break`，循环会直接停止，所以后面的元素不会再继续处理。因为返回的是固定长度数组，未赋值的位置会保留默认值 `0`，所以结果会像 `[1, 2, 3, 4, 0, 0, 0]`。

`listWithContinue()` 也使用 `for` 循环。当遇到数字 `5` 时，执行 `continue`，这一次循环被跳过，但后续的 `6` 和 `7` 仍然会继续执行。因为第 5 个位置没有被赋值，所以返回结果会保留一个默认值 `0`，类似 `[1, 2, 3, 4, 0, 6, 7]`。

`listWithWhile()` 展示了 `while` 的写法。只要条件 `i < list.length` 成立，循环就会一直继续。

## 注意事项

循环会消耗 Gas，所以链上逻辑里不要写过大的循环，也不要写无界循环。学习阶段可以先掌握语法和行为，再慢慢理解性能问题。
